# Python Tip Calculator

A simple Python tip calculator I built while learning Python before starting Computer Science at Carleton University.

Features

- Calculates tip using either:
  - A percentage of the bill
  - A fixed dollar amount
- Input validation using try/except
- Reusable functions
- Loops that prevent invalid input
- Calculates the final total including tip

Concepts Used

- Variables
- User input
- Functions 
- Parameters
- Return values
- while loops
- if / elif / else
- Error handling with try/except
- String formatting

Example

Bill Amount: $100

Tip Type (% or $): %

Tip Amount: 15

Tip: $15.00

Total: $115.00

Why I Built This?

I created this project while teaching myself Python before beginning COMP 1405 (Introduction to Computer Science I). The goal was to practice functions, loops, and user input while building something useful.
